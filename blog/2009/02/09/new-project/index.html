New Project WKTweet.com

Feb 9th, 2009

Today I created a web client to add Well-Known Text to Twitter.  It involved a lot of javascript and can be seen here.  Originally I had just planned to look for “POINT(” in a search to map everything but it quickly became apparent that wouldn’t work.  I’ve added a hashtag #WKT to each of the tweets from the website to make mapping easier.  My next mission it to actually map everything.

For an overview of WKT I suggest the Wikipedia article.  Also if you want to follow developments of this project follow WKTweet on Twitter.
